# 🌌 Indigo Horizon Dashboard

Este proyecto es un panel de control avanzado para la gestión de proyectos de arquitectura y especialistas.

## 🚀 Cómo ejecutar la aplicación (Windows)

He creado dos archivos de automatización para que no tengas que usar la terminal manualmente:

### 1. **Para desarrollar o ver cambios en vivo:**
Doble clic en: `INICIAR_APP.bat`
- Esto instalará las dependencias (si es la primera vez).
- Iniciará un servidor local.
- Abrirá tu navegador automáticamente en la dirección del proyecto.

### 2. **Para generar los archivos de producción (Publicar):**
Doble clic en: `CONSTRUIR_PROYECTO.bat`
- Esto generará una carpeta llamada `dist/`.
- El contenido de esa carpeta es lo que debes subir a tu servidor (ej. Hostinger, Netlify, Vercel).

---

## 🛠️ Requisitos previos

Para ejecutar este proyecto localmente, necesitas tener **Node.js** instalado en tu computadora.
- **Descárgalo aquí:** [https://nodejs.org/](https://nodejs.org/) (Se recomienda la versión **LTS**).

---

## 📁 Estructura del Proyecto

- `src/`: Contiene todo el código fuente (Componentes, Lógica, Estilos).
- `public/`: Archivos estáticos como el logo e imágenes.
- `package.json`: Configuración de dependencias y scripts.
- `tailwind.config.ts`: Configuración de los estilos visuales (colores, fuentes).

---

## 📝 Notas de Desarrollo

- La aplicación utiliza **React + TypeScript + Vite**.
- Los estilos están basados en **Tailwind CSS** para un diseño moderno y premium.
- Los iconos provienen de **Lucide React**.
